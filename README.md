- ### 首先进入shopbook-express项目根目录启动项目（cnpm install,cnpm run start）

- ### cnpm install 
- ### cnpm run dev
功能

```
- 登录 / 注销

- 全局功能
  - 国际化多语言
  - 动态侧边栏
  - 动态面包屑
  - 快捷导航(标签页) 
  - 自适应收缩侧边栏

- 組件
  - 头像上传
```
